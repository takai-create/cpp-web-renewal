Claude APIセットアップ手順とサンプルコード

【1. Claude API用パッケージのインストール】
ターミナルで以下を実行：
npm install anthropic

【2. Claude APIキーの取得と設定】
Anthropic公式サイトでAPIキーを取得し、.envファイルに追加：
ANTHROPIC_API_KEY=あなたのAPIキー

【3. Claude API呼び出し用サンプルコード（TypeScript例）】
import Anthropic from 'anthropic';

const anthropic = new Anthropic({
  apiKey: process.env.ANTHROPIC_API_KEY,
});

export async function askClaude(prompt: string) {
  const completion = await anthropic.messages.create({
    model: 'claude-3-opus-20240229',
    max_tokens: 1024,
    messages: [{ role: 'user', content: prompt }],
  });
  return completion.content[0].text;
}

【4. 既存のGPT呼び出し部分をClaude用に置換】
OpenAI（GPT）用のAPI呼び出し部分を、上記のClaude用関数に置き換えてください。

【5. 注意事項】
- .envファイルは.gitignoreに追加し、APIキーの漏洩を防ぐ
- 必要に応じてエラーハンドリングやレスポンス形式の調整を行う

この手順でClaude APIの利用が可能です。
