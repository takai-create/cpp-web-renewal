# Crowd Power Partners サイト構成検討プラン

## 目的
Crowd Power Partners のコーポレートサイトを、ブランドメッセージ「AIが動き、人が深める。」を中心に再設計する。抽象的な世界観で多角化事業を束ね、採用・提携・信頼構築を支援するサイト構成を作る。

## 対象ページ
- ホーム
- 企業/ブランドストーリー
- 事業紹介
  - DXコンサルティング
  - Talk Insights
  - コワーキング新潟弁天
- ニュース/インサイト
- 採用
- お問い合わせ
- プライバシーポリシー / 利用規約

## 主要セクション
- ヒーロー/メッセージビジュアル
- ブランドフィロソフィー
- 事業の柱（DX / AI / コワーキング）
- Talk Insights の価値提案
- 実績/信頼の証明
- ニュース・更新
- CTA（問い合わせ・資料請求・採用）

## ページ構成案

### 1. ホーム
- ヒーローメッセージ：「AIが動き、人が深める。」
- ブランドステートメント：コーポレートサイトとしての思想
- 事業の概観：DX / AI SaaS / コワーキング
- Talk Insights ハイライト
- コワーキングの特徴
- 連携・採用・問合せの CTA
- 最新ニュース

### 2. 企業/ブランドストーリー
- Vision / Mission / Philosophy
- 2013年設立からの歩み
- 東京・新潟の拠点
- 事業を束ねる「傘」の役割
- 主要メッセージの背景

### 3. 事業紹介
- 3つの柱を紹介
  - DXコンサルティング
  - Talk Insights
  - コワーキング新潟弁天
- 各事業の価値、ターゲット、事例
- Talk Insights の使い方と効果
- コワーキングの無人運営／拠点情報

### 4. ニュース/インサイト
- 企業ニュース
- 事業の活動報告
- イベント・メディア掲載
- ブログやコラム（必要に応じて）

### 5. 採用
- 企業カルチャー
- 働き方・価値観
- 募集ポジションと応募方法
- 代表メッセージ

### 6. お問い合わせ
- 事業相談フォーム
- 採用問い合わせフォーム
- パートナー・取材応募フォーム

### 7. 法務ページ
- プライバシーポリシー
- 利用規約

## ワイヤーフレーム検討のポイント
- トップにブランドメッセージと世界観を明確に提示
- 事業を「AI」「人」「リンク」という視点で整理
- Talk Insights はプロダクトよりも価値提案を先行
- コワーキングは「拠点とコミュニティ」の魅力を訴求
- 採用と提携の導線を常に意識

## 情報設計の検討項目
1. CTA の優先順位
   - 企業問合せ / 資料請求 / 採用応募
2. メッセージの表現レベル
   - 抽象的ブランドと具体的事業のバランス
3. ページ遷移設計
   - 事業紹介から問い合わせまでの導線
4. SEO キーワード
   - AI / DX / コワーキング / 新潟
5. コンテンツの更新頻度
   - ニュース・インサイト中心
