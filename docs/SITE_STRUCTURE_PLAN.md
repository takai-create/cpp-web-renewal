# Crowd Power Partners コーポレートサイト構成案

## 設計方針

### コンセプト
> **「AIが動き、人が深める。」**

プロダクト紹介サイトではなく、**思想で事業を語るコーポレートサイト**。
LayerX・ラクスルを参考に、多角化事業を抽象ビジョンで正当化し、
採用・提携・信頼構築を同時に果たす構造を目指す。

### ターゲットオーディエンス（優先順）
1. **提携候補・顧客企業** — 事業理解と信頼構築
2. **採用候補者** — カルチャーと成長機会の訴求
3. **投資家・メディア** — ビジョンと実績の提示

### 設計原則
- コーポレートサイトは「器」、各サービスLPは別設計
- 抽象（ビジョン）→ 具体（事業）→ 行動（CTA）の流れを全ページで維持
- 余白と書体で品格を出す（情報を詰め込まない）
- モバイルファースト、アニメーションは控えめ

## 各ページの役割と情報設計

### 1. ホーム `/`
**役割**: ブランドの第一印象。思想→事業→行動の導線起点。

| セクション | 内容 | 参考 |
|-----------|------|------|
| ヒーロー | 「AIが動き、人が深める。」＋抽象ビジュアル | LayerXトップ |
| フィロソフィー | 2〜3文でブランドの存在意義を語る | ラクスル「仕組みを変えれば」 |
| 事業の柱 | 3事業をカード形式で概観 | Sansan縦展開 |
| 数字で見る | 設立年・拠点数・導入実績等 | 信頼性の即時証明 |
| ニュース | 最新3件 | 更新性の担保 |
| CTA | 問い合わせ / 採用 / 資料請求 | 目的別導線 |

**詳細は HOME_PAGE_DESIGN.md を参照**

### 2. 私たちについて `/about`
**役割**: ビジョン・ミッション・沿革を通じて「なぜこの事業群なのか」を語る。

| セクション | 内容 |
|-----------|------|
| ビジョン/ミッション | 「AIが動き、人が深める。」の背景と意味 |
| 代表メッセージ | 信廣 宣浩の言葉で思想を語る（写真付き） |
| 沿革 | 2013年設立〜現在。事業の広がりを時系列で |
| 会社概要 | 社名・設立・所在地・代表者・事業内容 |
| 拠点情報 | 東京・新潟の2拠点体制 |

**設計意図**: LayerXのように「なぜ多角化するのか」をビジョンで正当化する。
代表の人格が見えることで、中小企業としての親近感と信頼を両立。

### 3. 事業紹介 `/business`
**役割**: 3事業を「傘」の下に束ねる。個別ページへの導線ハブ。

| セクション | 内容 |
|-----------|------|
| 事業フィロソフィー | 3事業を貫く思想を1文で |
| 事業カード×3 | DX / Talk Insights / コワーキング |
| 事業間のつながり | 「テクノロジー×現場×場」の連関図 |

| 対象 | 中小〜中堅企業 |
| CTA | 相談する |

#### 3-2. Talk Insights `/business/talk-insights`
| 要素 | 内容 |
|------|------|
| 見出し | 出会いを商談に変える。話すだけ。 |
| 概要 | 名刺交換後の音声メモ→自動フォローアップ |
| 特徴×3 | 音声録音 / AI要約 / Gmail自動送信 |
| 価格 | ¥800/月〜 |
| 対象 | ネットワーキング活発な中小企業オーナー・士業 |

**注**: コーポレートサイトでは価値提案を中心に。詳細機能は別LP。
