# ホームページ構成案とワイヤーフレーム

## 目的
Crowd Power Partners のホームページは、ブランドメッセージ「AIが動き、人が深める。」を起点に、事業の多様性と企業の信頼性を両立させることを目的とする。

## ホームページ構成案

### 1. ヒーローセクション
- メインメッセージ: 「AIが動き、人が深める。」
- サブテキスト: 事業の哲学とコーポレートの存在意義
- 主要CTA: 「お問い合わせ」「事業を見る」「採用情報」
- ビジュアルイメージ: AIと人の協調を象徴する抽象的な背景

### 2. ブランドステートメント
- ブランドの定義: 思想で事業を束ねるコーポレートサイト
- 会社のミッション・ビジョンを簡潔に表現
- 3事業をつなぐ軸としての「傘」の役割

### 3. 事業の柱
- 3つの事業をカード形式で紹介
  - DXコンサルティング
  - Talk Insights
  - コワーキング新潟弁天
- それぞれの価値とターゲットを短く説明
- 各事業ページへの導線ボタン

### 4. Talk Insights セクション
- 価値提案: 「出会いを商談に変える。話すだけ。」
- 主要機能: 音声メモ、Gmailフォローアップ、ネットワーキング強化
- ターゲット: 中小企業オーナー、士業、専門職
- CTA: 製品ページへ、体験や資料請求

### 5. コワーキング新潟弁天
- 無人運営・24時間利用の特徴
- 場所・設備・コミュニティ価値
- 地域拠点としての役割

### 6. 信頼/実績セクション
- 企業・事業の信頼を示す要素
- 事例、導入成果、メディア掲載など
- 代表的なキーワード: DX推進、AI活用、無人運営

### 7. 最新情報・ニュース
- 最新のアクティビティや更新を表示
- 企業の成長や発信の継続性を担保

### 8. CTAセクション
- 企業問合せ、採用応募、資料請求の3つの導線
- 各目的に応じた短い説明

### 9. フッター
- 会社概要リンク
- ニュース/採用/お問い合わせへの再導線
- SNSや所在地の表記

## ワイヤーフレーム

### レイアウト順
1. ヘッダー（ロゴ / ナビ / CTA）
2. ヒーロー
3. ブランドステートメント
4. 事業の柱
5. Talk Insights
6. コワーキング
7. 信頼/実績
8. 最新情報
9. CTAバナー
10. フッター

### 各ブロックの要素

#### ヘッダー
- ロゴ
- メニュー: ホーム / 企業・ブランド / 事業 / ニュース / 採用 / お問い合わせ
- CTA: 「資料請求」または「お問い合わせ」

#### ヒーロー
- 大見出し
- 補足説明
- 主要CTAボタン
- ビジュアルまたはグラフィック

#### ブランドステートメント
- シンプルな3〜4文
- 企業の存在意義と価値観
- 事業を束ねる「AI」「人」「リンク」のキーワード

#### 事業の柱
- 3つ並列カード
- アイコンまたは画像
- 1行キャッチコピー
- 詳細リンク

#### Talk Insights
- 見出し + サブテキスト
- 3つの特徴（音声・自動化・ネットワーキング）
- CTAボタン

#### コワーキング
- 無人運営訴求
- 24時間利用、拠点の魅力
- 施設イメージと利用シーン

#### 信頼/実績
- 3〜4の実績要素
- メディア掲載、導入事例、数値

#### 最新情報
- 最新ニュース3件
- カテゴリ/タグラベル

#### CTAバナー
- 目的別CTAを横並びまたは縦並び
- 「お問い合わせ」「採用」「資料請求」

#### フッター
- 会社情報
- ページリンク
- 連絡先
- コピーライト
