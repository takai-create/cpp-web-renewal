## 会社・事業概要

- **設立**：2013年
- **拠点**：東京・新潟
- **事業領域**：
  1. DXコンサルティング（GWSリセラー含む）
  2. AI SaaS「Talk Insights」（旧称：First Touch）
  3. コワーキングスペース「コワーキング新潟弁天」（24時間無人運営）

### Talk Insights について
- 名刺交換後に音声メモを録音 → 自動でGmailにフォローアップメール下書きを生成
- ターゲット：ネットワーキング活発な中小企業オーナー・士業・専門職
- 価格：¥800/月〜
- キャッチコピー：「出会いを商談に変える。話すだけ。」
- 技術スタック：Next.js / Neon PostgreSQL / OpenRouter（Claude Sonnet）
