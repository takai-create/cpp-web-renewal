# ホームページ デザイン仕様

## 設計コンセプト

**参考モデル**: LayerX × ラクスル のハイブリッド
- LayerX的: 抽象ビジョンで多角事業を束ねる構造
- ラクスル的: 「仕組みを変える」型の力強い一文コピー

**トーン**: 洗練・知的・信頼感。派手さより品格。
**カラー**: ダークネイビー基調 × ホワイト × アクセントブルー

---

## カラーパレット

| 用途 | 色 | コード |
|------|-----|--------|
| プライマリ（背景・テキスト） | ダークネイビー | `#0A1628` |
| セカンダリ（背景） | ホワイト | `#FFFFFF` |
| アクセント | ブライトブルー | `#2563EB` |
| テキスト（本文） | ダークグレー | `#374151` |
| テキスト（薄） | ミディアムグレー | `#6B7280` |
| 背景（交互セクション） | ライトグレー | `#F9FAFB` |
| ボーダー | ライトボーダー | `#E5E7EB` |

## タイポグラフィ

| 要素 | フォント | サイズ（PC / SP） | ウェイト |
|------|---------|------------------|---------|
| ヒーロー見出し | Noto Sans JP | 56px / 32px | Bold (700) |
| セクション見出し | Noto Sans JP | 36px / 24px | Bold (700) |
| サブ見出し | Noto Sans JP | 20px / 16px | Medium (500) |
| 本文 | Noto Sans JP | 16px / 14px | Regular (400) |
| ナビ | Inter + Noto Sans JP | 14px / 14px | Medium (500) |
| ボタン | Noto Sans JP | 14px / 14px | Bold (700) |

**英字**: Inter をプライマリに、日本語は Noto Sans JP にフォールバック。

## セクション詳細設計

### ━━━ Section 1: ヘッダー ━━━

```
┌─────────────────────────────────────────────────────────┐
│  [Logo]    私たちについて  事業紹介▼  ニュース  採用  [お問い合わせ] │
└─────────────────────────────────────────────────────────┘
```

- 固定ヘッダー（スクロール時: 背景 `rgba(255,255,255,0.9)` + backdrop-blur）
- ロゴ: 左寄せ
- ナビ: 中央〜右寄せ
- 「お問い合わせ」: アクセントブルーのボタン、角丸8px
- 高さ: 72px（PC）、60px（SP）

**レイアウト**:

**アニメーション**:

## インタラクション・アニメーション

| 要素 | アニメーション | タイミング |
|------|-------------|----------|
| ヒーローテキスト | フェードイン（上→下） | ページロード 0.3s |
| ヒーローCTA | フェードイン | 0.6s遅延 |
| セクション見出し | フェードイン | スクロールイン |
| 事業カード | 順次フェードイン | スクロールイン、0.2s間隔 |
| 数字 | カウントアップ | スクロールイン |
| カードホバー | 上昇2px + 影強化 | ホバー 0.2s ease |
| ボタンホバー | 背景色変化 | ホバー 0.15s ease |
| ヘッダー | 背景ぼかし追加 | スクロール位置 > 100px |

**原則**: アニメーションは「気づかない程度」に。過度な演出は避ける。

## 技術仕様（参考）

- **フレームワーク**: Next.js (App Router)
- **スタイリング**: Tailwind CSS
- **フォント**: Google Fonts（Inter + Noto Sans JP）
- **アイコン**: Lucide React
- **アニメーション**: CSS transitions + Intersection Observer
- **ホスティング**: Vercel（推奨）
